# Claude Code 小工具集

Windows 專用的 Claude Code CLI 實用工具集合。每個工具為獨立資料夾，包含自己的 README、安裝腳本和開發文件。

## 工具列表

| 工具 | 說明 |
|------|------|
| [claude-code-screenshot-paste](./claude-code-screenshot-paste/) | 讓 Claude Code 支援 Ctrl+V 貼上截圖，自動將剪貼簿圖片存為 PNG 並替換為檔案路徑 |

## 系統需求

- Windows 10 / 11
- PowerShell 5.1（Windows 内建）
- Claude Code CLI
